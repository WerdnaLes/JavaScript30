## Noteworthy points:

### Make canvas width and hight match your window dimensions:

```javascript
canvas.width = window.innerWidth;
canvas.height = window.innerHeight;

```
### Shorthand of assigning some variables:

```javascript
[lastX, lastY] = [e.offsetX, e.offsetY]; // shorthand
//   lastX = e.offsetX;
//   lastY = e.offsetY;
```
